# Simply Smarter Ltd - Landing Page PRD

## Project Overview
Corporate landing page for Simply Smarter Ltd - a wealth management and AI-driven asset security company.

## Original Problem Statement
Build a lean, high-performance, single-page corporate landing page for Simply Smarter Ltd with professional, modernist, minimalist design using high-contrast white text on deep charcoal/black background.

## User Personas
- **Professional Investors**: Individuals interested in AI-driven wealth management tools
- **Crypto/Precious Metals Enthusiasts**: Users building diverse asset portfolios (Bitcoin, gold, metals)
- **Privacy-Conscious Users**: Those seeking secure, private asset management solutions

## Core Requirements (Static)
- [x] SVG geometric 'S' logo in top left
- [x] "SIMPLY SMARTER LTD" title center-right of logo
- [x] Contact email: admin@simplysmarter.io
- [x] No buttons in header (clean layout)
- [x] Mission Statement section
- [x] Vision Statement section
- [x] Future Projects "Coming Soon" placeholder
- [x] Footer with copyright
- [x] Subtle scroll animations (Framer Motion)
- [x] No social media links
- [x] No SSW3.io references
- [x] No personal/business details (privacy-focused)

## Color Palette
- Background: #050505 (deep charcoal)
- Text Primary: #FFFFFF (white)
- Text Secondary: #A1A1AA (neutral gray)
- Accent: #F59E0B (gold/orange - reserved for stacking icons only)
- Border: #27272A
